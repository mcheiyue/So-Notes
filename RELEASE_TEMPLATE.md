# SoNotes v1.1.9

### ✨ 新特性 (Features)
*   **一键复制 (One-click Copy)**
    > 还在手动选中文字？现在只需轻轻一点便签顶部的复制按钮，即可瞬间提取内容。
    > *   **极简反馈**: 图标即时变为对钩，无需任何弹窗打扰，操作行云流水。
    > *   **智能格式**: 若便签有标题，会自动以 `标题 + 换行 + 内容` 的格式复制，保持信息完整。
*   **原生行为守卫 (Native Guard)**
    > 为应用穿上了一层防弹衣。我们拦截了所有可能破坏体验的浏览器原生快捷键：
    > *   **防误触**: 彻底禁用了 `F5` / `Ctrl+R` 刷新、`Ctrl+S` 保存网页、`Ctrl+F` 查找栏等“出戏”的浏览器行为。
    > *   **防破坏**: 锁死了 `Ctrl + 滚轮` 缩放，防止误操作破坏精心布局的画布。
    > *   **纯净右键**: 在非编辑区域屏蔽了原生右键菜单，体验更接近原生应用。

### 🚀 优化 (Optimizations)
*   **拖拽性能革命 (Performance)**
    > 即使画布上有 100+ 便签，拖拽也如丝般顺滑。
    > *   **CSS 硬件加速**: 摒弃了昂贵的 React 全局状态渲染，转而使用 CSS 选择器 (`body.is-dragging`) 来控制界面交互。
    > *   **O(1) 复杂度**: 实现了拖拽时的“零重绘”机制，无论便签多少，性能恒定。
*   **布局微调 (Layout)**
    > 修复了便签头部按钮不对称导致标题无法居中的视觉瑕疵。现在无论左右按钮如何增减，标题永远稳居 C 位。
