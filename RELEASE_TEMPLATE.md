# SoNotes v1.2.0

### ✨ 新特性 (Features)
*   **深色模式 (Dark Mode)**
    > 为夜间工作者精心打造的沉浸式体验。
    > *   **暗夜宝石 (Dark Gemstone)**: 告别沉闷的死黑。我们采用了一套全新的“宝石色”映射策略，让五彩斑斓的便签在深色背景下依然通透、鲜艳，宛如暗夜中发光的宝石。
    > *   **玻璃拟态 (Glassmorphism)**: 全面引入高级毛玻璃效果，配合微光边框，让每一个 UI 元素都悬浮于背景之上，层次分明，质感细腻。
    > *   **智能切换**: 支持“浅色”、“深色”及“跟随系统”三种模式，无缝融入您的操作系统环境。
*   **设置中心 (Settings Center)**
    > 重新设计的 Dock 设置菜单，现已支持二级导航。
    > *   **层级清晰**: 将“主题模式”与“数据管理”分层展示，操作路径更直观。
    > *   **数据管理**: 集中管理全量备份、看板导出与数据恢复，安全感倍增。

### 🐛 问题修复 (Bug Fixes)
*   **交互修复**: 修复了在平移模式 (Pan Mode) 下，点击便签仍会触发选中，导致无法拖动画布的问题。现在平移操作更加行云流水。
*   **小地图层级**: 修复了小地图可能被便签遮挡的层级问题，确保概览视图始终可见。
*   **拖拽残留**: 修复了通过小地图快速跳转视口后，可能导致鼠标陷入“拖拽锁定”状态的交互 Bug。

### 🚀 优化 (Optimizations)
*   **视觉升级**: 全局引入语义化颜色系统，统一了右键菜单、搜索框 (Spotlight)、废纸篓等所有辅助界面的视觉语言。
*   **防闪烁**: 优化了应用启动时的 CSS 注入逻辑，彻底根除深色模式下的“白屏闪烁 (FOUC)”现象。
*   **搜索增强**: 恢复并优化了 Spotlight 的权重搜索算法，让搜索结果更精准地命中您的目标。
