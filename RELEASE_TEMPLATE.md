# 🎉 SoNotes v1.0.0 - Initial Release

**SoNotes (随心记)** 是一款专为 Windows 11 设计的极简托盘便签应用。它不仅拥有原生的 Mica 磨砂质感，更提供了“零摩擦”的记录体验。

## ✨ 核心特性 (Highlights)

*   **🎨 原生美学 (Native Fluent Design)**
    *   完美复刻 Windows 11 控制中心风格，拥有细腻的圆角、阴影和 Mica 材质背景。
    *   提供 6 种精心调配的莫兰迪色系便签，清爽护眼。
*   **🚀 极致轻量 (Tray-First)**
    *   常驻系统托盘，点击图标即开即用，点击桌面即自动收回。
    *   面板悬浮于任务栏上方，支持自由拖拽大小。
*   **⚡ 零摩擦交互 (Zero Friction)**
    *   双击任意空白处即可新建便签。
    *   输入内容自动保存（原子写入 + WAL 双重保险），永不丢数据。
*   **🛡️ 隐私与安全 (Local First)**
    *   所有数据仅存储在本地 JSON 文件中，无云端上传，安全可控。

## 📦 下载与安装 (Installation)

本版本为 **绿色单文件版 (Portable)**，无需安装：

1.  在下方 **Assets** 区域找到 `so-notes.exe`。
2.  下载后直接双击运行。
3.  建议将应用固定到开机启动（需手动创建快捷方式到 Startup 文件夹）。
